# Frontend

O frontend do KukuGest vive nesta pasta e é uma aplicação Next.js 14.

Para evitar documentação duplicada, a fonte de verdade está na raiz do repositório:

- `../README.md` para visão geral e setup
- `../DEPLOYMENT.md` para deploy
- `../DATABASE_SETUP.md` para base de dados

## Comandos

```bash
npm install
npm run dev
npm run build
```
