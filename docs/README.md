## Estrutura

- `overview/`: briefing técnico e contexto para avaliação do projeto
- `product/`: visão funcional e lógica do sistema
- `setup/`: configuração técnica local e base de dados
- `operations/`: deploy e operação dos ambientes
- `standards/`: convenções de engenharia, arquitetura e qualidade
- `modules/`: documentação especializada por módulo
- `integrations/`: documentação específica de integrações externas
- `examples/`: ficheiros de apoio e amostras para teste manual
